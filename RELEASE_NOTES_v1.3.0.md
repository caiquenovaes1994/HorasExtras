# Notas de Lançamento - v1.3.0

## Conformidade Legal & LGPD

Esta versão implementa a obrigatoriedade do aceite dos **Termos de Uso e Política de Privacidade**, garantindo a conformidade com a **Lei Geral de Proteção de Dados (LGPD)**.

## O que há de novo

* **Trava de Segurança LGPD**:
  * Bloqueio completo do sistema para usuários que não aceitaram formalmente os termos.
  * Tela de aceite obrigatório exibida imediatamente após o login ou troca de senha.
  * Registro auditável de data, hora e fuso de Brasília (`America/Sao_Paulo`) do momento do aceite.
* **Transparência de Dados**:
  * Resumo simplificado explicando a finalidade da coleta de dados de jornada.
  * Detalhes sobre a segurança e criptografia AES-256 aplicada a dados financeiros.
